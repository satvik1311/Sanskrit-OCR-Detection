# Sanskrit OCR (Text Detection)

This project implements a CRAFT-inspired deep learning model for detecting Sanskrit text in images.

## 🚀 Features
- Custom CNN backbone (TensorFlow)
- Bounding box detection (CRAFT-style)
- Annotation processing pipeline (Label Studio)
- IoU metric & custom loss function
- Image preprocessing & target generation

## 🧠 Tech Stack
- Python
- TensorFlow / Keras
- OpenCV
- NumPy

## 📂 Pipeline
1. Image preprocessing
2. Annotation parsing
3. Target generation
4. Model training (text detection)

## ⚠️ Note
This project focuses on **text detection**, not full OCR (text recognition not included).

- Add CRNN / Transformer for text recognition
- Deploy as API
